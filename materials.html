If you unzip projectTweet.zip, there will be a number of directories:

  • analysis/
    contains our files from data analysis, this also contains the Microsoft Excel VBA program
  • collection/
    contains our files from the data collection process, we created programs for Twitter's REST and Streaming API, both can be found here
  • csv/
    contains our files from the csv exports after the data collection
  • dump/
    contains our raw data dump from the data collection process
  • images/
    contains our rendered images and screenshots that we created for the final report
  • site/
    contains all the code for our project website
